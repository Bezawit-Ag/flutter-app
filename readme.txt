Overview

A Flutter mobile app for discovering, creating, and planning recipes. Users can search, filter, save favorites, assign recipes to days, and generate grocery lists.

Features

Search & Filter: by name, ingredients, meal type, difficulty, time, calories, tags

Recipe Details: image, title, time, calories, difficulty, tags, favorite icon

Favorites: save/unsave recipes, offline access

Create Recipe: add title, image, meal type, difficulty, servings, nutrition, tags

Meal Planning: assign recipes to days, weekly view, generate grocery list

Grocery List: auto-group ingredients by category, check-off items, show progress
